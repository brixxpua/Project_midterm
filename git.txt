to initialize:
git init

To add a file:
git add <filename>

To check of tree:
git status

For message to know what was changed:
git commit -m "message"

To check log:
git log

To add many/all file:
git add .


To Delete a File:
need to go to staging then repository

git status --> git add . --> git commit -m 'Deleted <filename>" --> git log

to restore a file:
git reset --hard/--soft hash
git reset --hard HEAD~1

difference between hard and soft 
 hard - hard reset (ubos ang code)
 soft - notifying

To add and switch branch:
git checkout -b "nameofbranch"
git branch
git switch "nameofbranch"


Merging:

git merge "name of branch"
